🔗 API INTEGRATION GUIDE
═════════════════════════════════════════════════════════════════════════════

JSONBin.io Setup (Recommended):

1. Create Account
   → https://jsonbin.io
   → Sign up (free tier available)

2. Create a Bin
   → Click "Create Bin"
   → Name: "DebtEase-Dataset"
   → Paste merged_dataset.json content
   → Copy and save your BIN_ID

3. Get API Key
   → Settings → API Key

4. Use in Dashboard
   → Open index_api.html
   → Select "API Integration" radio button
   → Enter Bin ID and API Key
   → Click "Connect to API"
   → Done! Data fetches from remote API

Alternative Services:
  • MockAPI.io (https://mockapi.io/)
  • GitHub Raw URLs
  • Your own server

🔧 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Q: Dashboard shows "Loading data..." but doesn't display
A: Check browser console (F12 → Console) for errors
   Verify JSON files exist in the same folder

Q: Port 8000 already in use
A: python -m http.server 8001  [or any free port]

Q: API connection fails
A: Verify internet connection
   Check Bin ID and API Key are correct
   Ensure JSONBin.io service is online

Q: How to run for a new dataset?
A: 1. Replace CSVs in Dataset/ folder
   2. python merge_csv.py
   3. python convert_to_json.py
   4. python debt_ease_algorithm.py
   5. refresh browser
